Есть возможность добавить клиента с любым именем.
Для этого требуется, чтобы в реестре, в графе имени клиента
было напечтано ТОЧНО такое же имя, как имя файла озвучки:
Example:
Клиент "GDS"
Называем файл озвучки GDS.wav и сохраняем в папку voice.
